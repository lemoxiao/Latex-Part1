## 环境要求
* 操作系统：Linux/Windows
* 编译器：Luatex,Bibtex
* 宏包：
    - 中文支持：ctex
    - 页面调整：fancyhdr,geometry
    - 数学公式：amsmath,amsthm,amsfonts,amssymb,bm
    - 超链接：cite,hyperref
    - 图表绘制：float,graphicx,tikz,pgfplots,array
    - 代码插入：minted

# Windows

## 工具
* Convert.bat   将目录下png文件转换为eps文件
* Created.bat   实现tex文件到pdf的自动转换
* Clear.bat     清理临时文件
